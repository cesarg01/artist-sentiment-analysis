About

The site analyzes the sentiment (mood) of the albums from the user inputted artist. One will be able to see how the mood changes from album to album and where each song is in the scale.

Where does the data come from?

The data used comes from the Spotify API valence value which Spotify describes it as, "A measure from 0.0 to 1.0 describing the musical positiveness conveyed by a track. Tracks with high valence sound more positive (e.g. happy, cheerful, euphoric), while tracks with low valence sound more negative (e.g. sad, depressed, angry)." The polarity of each song is analyzed using TextBlob, a Python library for processing textual data, to get the sentiment value which basically means determining the attitude or emotion of the artist.
